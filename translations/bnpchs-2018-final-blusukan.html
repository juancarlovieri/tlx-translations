Description

Ivo plans to make an Oivworld digital clock. Ivo knows that in Oivworld, time is not represented as hours and minutes, but maj and netim. Fortunately, Ivo has a guide book about the time cycle in Oivworld, which is:

  • one maj is equal to $$$A$$$ netim;
  • the time format in Oivworld is $$$N$$$ maj $$$M$$$ netim, where $$$(0 \le M < A$$$.

Ivo asks you, a dweller of Oivworld. If $$$X$$$ netim has passed, what is the time format in $$$N$$$ maj $$$M$$$ netim?

Input Format

A X

Output Format

One line containing two integers $$$N$$$ and $$$M$$$, representing the time has passed in Oivworld for $$$N$$$ maj $$$M$$$ netim.

Sample Input 1

2 5

Sample Output 1

2 1

Sample Input 2

4 20

Sample Output 2

5 0

Sample Input 3

9 6

Sample Output 3

0 6

Explanation

In the first sample, 5 netim is equal to 2 maj and 1 netim.

Constraints

  • $$$1 ≤ A, X ≤ 100$$$.
